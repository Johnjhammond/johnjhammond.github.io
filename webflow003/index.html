Interactivity experiments

I've been using Webflow with After Effects / Bodymovin to get animations to interact with user input.

This is a responsive page, I've created fluid layouts that work on desktop, tablet and mobile.

Think of these interactions as ingredients that I can use with interesting animations to create a compelling user experience.

Sync to scroll

This Lottie animation is synced to the vertical viewport scrolling. It starts as it enters the frame and finished as it leaves - with offset applied.

Parallax

Two Lotties, each with position animation linked to scroll, but at different rates to create the effect of parallax / depth.

Webflow (the cloud based app) has really helped me out! It handles the back end code, however, I still just spent a whole evening getting this parallax to work. I understand HTML/CSS a lot better now... Static, flex box, grid, absolute, relative... playing with the Webflow UI really helps me understand it all.

Notes: Parent Div was set to Grid, 1x1, with each child (Lottie) set to manual, span 1x1 so that they overlap on top of each other.

parallax + full screen

This one uses 3 layers animating at different speeds with scroll, creating a parallax effect. Additionally, there's a full screen wind layer, triggered when you arrive at this point on the page.

I've added a bit of animation with the tree (leaves blow out) that is timed with the full screen wind lines animation... Now that I understand what's possible with the Web technology, I can really push things as the planning and animating stages to make my various elements, layers, etc sync up... the possibilities are pretty awesome (in my head).
