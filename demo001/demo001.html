My First Lottie Animation


Above is an animated shape made in After Effects and exported as a Lottie (.json) file with the Bodymovin plugin.

I created this Webpage using HTML and CSS, and used the Lottie Web Player, which is a block of JavaScript code, to embed the Lottie Animation.

Why do this? Why not use a .gif or MPG4 video?

Lottie animations are not pixel based, they are code based, in the form of an SVG (scalable vector graphics).

This means that they are lightweight, with small file sizes, scalable and can be responsive (try scaling this browser window).

They are great for building animation into interactive apps, on the Web or for mobile or desktop apps.

You can add lots of cool interactivty via code and make rich interactive user interfaces.

I'm experimenting with this as a way to create a solution for Leigh Woods.
